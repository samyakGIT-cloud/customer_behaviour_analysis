📊 Sales Data Analytics Project
📌 Overview

This project demonstrates a complete data analytics workflow using Python, SQL, and Power BI.
The goal of the project is to analyze a sales dataset, uncover meaningful insights, and present the findings through an interactive dashboard and report.

The project includes data loading, exploratory data analysis (EDA), data cleaning, SQL analysis, dashboard creation, and final reporting.

📂 Dataset

The dataset used in this project contains information related to sales transactions such as:

Order ID

Customer details

Product information

Sales amount

Quantity

Region / Location

Order date

The dataset is used to analyze sales trends, customer behavior, and product performance.

🛠 Tools & Technologies

The following tools were used in this project:

Python – Data loading, cleaning, and exploratory data analysis

Pandas & NumPy – Data manipulation and analysis

Matplotlib / Seaborn – Data visualization during EDA

PostgreSQL – Running SQL queries for business insights

Power BI – Creating an interactive sales dashboard

Gamma – Creating the final presentation (PPT)

Jupyter Notebook – Executing Python analysis

⚙️ Project Steps
1️⃣ Data Loading

Imported dataset using Python (Pandas).

Loaded the data into a Jupyter Notebook for analysis.

2️⃣ Exploratory Data Analysis (EDA)

Checked dataset structure and data types

Analyzed summary statistics

Identified trends and patterns in sales data

Created visualizations to understand data distribution

3️⃣ Data Cleaning

Removed duplicate records

Handled missing values

Corrected inconsistent data formats

Prepared the dataset for analysis

4️⃣ SQL Analysis (PostgreSQL)

The cleaned dataset was loaded into PostgreSQL to perform SQL analysis.

Example queries performed:

Total sales by region

Top selling products

Monthly sales trends

Customer purchase behavior

📊 Power BI Dashboard

An interactive Power BI dashboard was created to visualize key business insights.

Dashboard includes:

Total sales overview

Sales by region

Top products by revenue

Monthly sales trends

Customer segmentation

The dashboard helps stakeholders quickly understand sales performance and patterns.

📈 Results & Insights

Some key insights from the analysis:

Identified top-performing products and regions

Observed seasonal sales trends

Discovered high-value customer segments

Found opportunities to improve sales in low-performing regions

These insights can help businesses optimize strategy and improve decision making.

📑 Report & Presentation

A detailed report summarizing the analysis was created.

A presentation (PPT) was generated using Gamma to explain insights in a clear and visual format.

▶️ How to Run the Project

Clone the repository

git remote set-url origin https://github.com/samyakGIT-cloud/customer_behaviour_analysis.git

Navigate to the project folder

cd data-analytics-project

Install required Python libraries

pip install pandas numpy matplotlib seaborn psycopg2

Run the Jupyter Notebook to perform:

Data loading

EDA

Data cleaning

Load the cleaned dataset into PostgreSQL and run SQL queries.

Open the Power BI dashboard file (.pbix) to explore the visualizations.
